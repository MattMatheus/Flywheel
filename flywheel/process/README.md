# Process Docs

These docs define the generic Flywheel workflow contract.

Stage checklists, exit gates, and forbidden actions live in one place:
`flywheel/stage_contracts.yaml`. The docs here cover the cross-stage process
rules that the contracts cannot express.

## Core Docs
- `CYCLE_NAVIGATION.md`
- `HANDOFF_EXPECTATIONS.md`
- `VERIFICATION_PATTERNS.md`
- `BRANCH_AND_COMMIT_INVARIANTS.md`

All workflow locations are resolved through `flywheel.yaml`.
